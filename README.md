# Hangman project

## Description

Play Hangman, by spelling out the concealed word before the character walks the plank

## Contains

The website contains the following:
* **Scores tally** - tracking your number of wins and losses
* **Game board**:
    * **Concealed Word** - that is revealed letter-by-letter as you guess letters correctly
    * **Keyboard Interface** - guess letters by either clicking the on-screen buttons or typing on your keyboard
    * **Progressive Image** - with each incorrectly guessed letter, watch the character make their way along the plank until the perilous jump
* **Previous words** list
* **Play Again** - on game end, see the option to replay automatically appear

## Features

* **Mobile-First Design** - this website has been built with a resposive, mobile-first text
* **Data persistence** - Scores and previously played words withstand page refresh
* **Replayability** - Resets game board and dynamically selects new words

## Technologies Used

* JavaScript
* HTML
